```text
(briefing about the evaluation then begin the test)
I : Seberapa sering anda menggunakan internet? Kira kira berapa jam sehari?
X : Sering sekali, lebih dari 10 jam per harinya.
I : Apa saja website yang sering dikunjungi?
X : Youtube, Social Media, Stackoverflow, mungkin itu.
I : Pernahkah Anda mengunjungi website yang memiliki bad design?
X : Pernah
I : Bagaimana pendapatmu?
X : Pertama, bikin bingung yang makai walaupun mereka harusnya punya fungsi-
    fungsi bagus yang harusnya mempermudah pengguna serta memperbagus image
    mereka, tetapi malah menjadi counter-product, memperburuk kondisi yang
    ada.
I : Berarti Anda kesulitan kalau menggunakan website tersebut?
X : Ya
I : Berlanjut ke pertanyaan spesifik, Anda tau Capture The Flag? Silakan
    diterangkan ke saya sepengetahuan Anda.
X : Ya, jadi CTF itu intinya website yang kita bisa mengerjakan soal-soal
    terkait dengan keamanan jaringan, dari trivia, eksploitasi web.
    reverse engineering, kriptografi, dan lain lain.
I : Pernahkah Anda menggunakan platform web CTF?
X : Pernah, CTFTime. Terakhir memakai semester lalu.
I : Selanjutya adalah instruksi, pertama cobalah untuk masuk ke dashboard.
X : (Mengisi form registrasi dan login)
I : Sebut dan ucapkan 5 problem teratas saat ini.
X : (Menyebutkan 5 problem teratas)
I : Coba download sebuah file dalam suatu problem.
X : (Mencoba membuka problem yang memiliki logo file, mengklik hyperlink
    untuk mendownload file, kemudian menunjukkan file yang didownload)
I : Selanjutnya solve 1 problem, bebas.
X : (Solve problem yang dibuka pada instruksi sebelumnya dengan melihat
    file didalamnya terdapat flag)
I : Buat 1 problem dan sertakan 1 file didalamya.
X : (Hover problems, mengklik create problem, mengisikan data dan file
    sukses add problem baru)
X : Wooh.. (Terkejut karena sukses membuat problem dan langsung tampil)
I : Tampilkan problem yang habis dibuat
X : (Menampilkan problem)
I : Selanjutnya, edit problem yang habis dibuat.
X : (Membuka my problems, mengklik edit, sukses edit problem)
I : Bisa ditunjukkan?

I : Coba sebutkan username dengan skor tertinggi.
X : Username dengan skor tertinggi adalah Rahma, dengan skor 10000.
I : Tunjukkan submission histori Anda.

    submission history)
I : Bisa ditunjukkan problem no. 5 itu yang mana?
X : (menunjukkan problem nomor 5 di list problem) Ini.
I : Sekarang coba log out dari sistem.
X : Ya baik. Terimakasih.
I : Terakhir, apakah ada komentar dari test sistem ini?

    yang memakai sistem ini, gak terlalu bingung.
(end)
```
